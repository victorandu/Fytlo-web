# Fytlo Marketing Site

Marketing site and waitlist for [[Fytlo]] — an AI-powered virtual try-on platform focused on realistic fit visualization. Part of [[Pausora Labs]].

## This repo contains
- Landing page
- Waitlist flow
- Frontend experiments (`/`, `/privacy`, `/terms`)

## Tech stack
- Next.js 14 (App Router), TypeScript, TailwindCSS, pnpm
- Dark Liquid Glass design system
